# Android Image Modifier

## About

This script helps in automating compress/decompress of system.new.dat files.

## Requirements 

 * python 
         ```
   Arch: pacman -S python
         ```
 * img2simg 
           ```
     Arch: pacman -S aur/simg-tools
           ```
 *  git clone $this_repo --recursive


Usage

```
chmod +x start.sh
./start.sh
```
